This demo is isolated from other demos.

Agents may edit only files under this directory unless explicitly instructed.

Build command:
```sh
xcodebuild -project DemoServerSim.xcodeproj -scheme DemoServerSim -destination 'generic/platform=iOS' build
```

Serve-sim run command:
```sh
./scripts/run-serve-sim.sh
```
